# TailwindCSS Practice Projects

This repo is my playground for learning **TailwindCSS**.  
Here I experiment with utility classes, components, responsive layouts, and small JS interactions.

## What's inside
- `/modal/` – Experimenting with modal components  
- `/cards/` – Card designs and variations  
- `/navigation/` – Navbar experiments  
- `/layouts/` – Grid & Flexbox layouts  
- `/components/` – Reusable UI pieces  

## Tech
- HTML5
- TailwindCSS
- Vanilla JS (sometimes)

## Why this repo
- Practice Tailwind utilities  
- Build small UI components from scratch  
- Learn responsive design patterns  
- Improve speed and cleanliness in frontend styling  

> Note: Not production code, just my personal experiments.
